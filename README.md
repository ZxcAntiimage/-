# EnerflexCBD - Landing Page

## 📋 Описание проекта

Landing page для продукта **EnerflexCBD** - инновационной добавки для здоровья суставов. Проект представляет собой одностраничный сайт с адаптивным дизайном, модальным окном для оформления заказа и интеграцией с API для отправки лидов.

## 🚀 Технологии

- **HTML5** - семантическая верстка
- **CSS3 / Tailwind CSS** - стилизация и адаптивный дизайн
- **JavaScript (Vanilla)** - интерактивность без фреймворков
- **PHP 7.4+** - бэкенд для обработки заказов
- **BEM** - методология именования классов

## 📱 Адаптивность

Проект полностью адаптивен под:
- **Мобильные устройства** (до 768px) - меню-бургер, колоночная верстка
- **Планшеты** (768px - 1024px)
- **Десктопы** (1024px+) - горизонтальное расположение блоков

## 🎯 Функциональность

### Основные блоки:
- **Header** - навигация с бургер-меню на мобильных
- **Hero section** - заголовок, иконки и список проблемных зон
- **Price card** - старая/новая цена + кнопка заказа
- **Focus grid** - сетка проблемных зон
- **Benefits** - преимущества продукта
- **Expert opinion** - мнение эксперта с пошаговым процессом
- **Efficacy** - активные компоненты с сертификатами
- **FAQ** - аккордеон с вопросами и ответами
- **Reviews** - карусель отзывов с пагинацией
- **Footer** - повторный блок с ценой и формой заказа

### Модальное окно:
- Открывается при клике на любую кнопку **Order**
- Затемненный фон с блюром (backdrop-filter)
- Форма с полями: имя, телефон
- Отправка данных через AJAX на `order.php`
- Валидация полей
- Сообщения об успехе/ошибке
- Закрытие по клику на оверлей, крестик или клавишу ESC

## 🔌 API интеграция

### Обработчик заказов (`order.php`)

**Метод:** POST

**Параметры:**
```json
{
  "name": "Имя клиента",
  "phone": "Телефон клиента",
  "sub1": "clickid (опционально)",
  "sub2": "utm_medium (опционально)"
}
```

## ⚡ JavaScript модули
1. Мобильное меню

   - Открытие/закрытие бургер-меню

   - Закрытие по клику вне меню или на ссылку

2. Карусель отзывов

   - Показ одного отзыва за раз

   - Переключение по стрелкам и пагинации

   - Адаптация при ресайзе окна

3. FAQ аккордеон

   - Открытие/закрытие ответов по клику

   - Анимация плавного раскрытия

   - Закрытие предыдущего при открытии нового

4. Модальное окно

   - Открытие по кнопке Order

   - Отправка формы с валидацией

   - Обработка ответа от PHP
